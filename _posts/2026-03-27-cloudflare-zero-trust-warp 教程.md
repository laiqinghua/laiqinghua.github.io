# Cloudflare Zero Trust 免费 WARP 配置教程

今天讲一下互联网"大善人"Cloudflare 旗下的 Zero Trust，通过 WARP 客户端构建安全连接，让你能安全访问全球网络。

小编已把完整的注册、安装和配置流程整理好，你照着步骤操作即可。

## 一、注册 Cloudflare Zero Trust 账户

### 1.1 访问注册页面

浏览器打开这个网址：

```
https://dash.cloudflare.com/sign-up/zero-trust
```

### 1.2 选择注册方式

注册方式很灵活：

- 使用 Google 账号快速登录
- 使用 Apple 账号快速登录
- 通过邮箱注册

![注册 Cloudflare Zero Trust](/assets/images/post/2576c5ef4554f0ad33b9d214661cf24e.jpg)

### 1.3 创建团队名称

登录成功后，在左侧菜单找到 **Zero Trust** 入口并点击进入。

如果是第一次使用，会看到欢迎界面，点击"开始使用"。

接下来创建一个团队名称，这个名称是你登录客户端时的 Zero Trust 域名标识，在登录客户端时会用到。名称可以自定义，如果被占用就换一个即可。

![创建团队名称](/assets/images/post/113e672f79edb3edf048228c1e381714.jpg)

### 1.4 选择免费套餐

创建完成后，选择免费套餐（0 元方案即可）。

**免费版额度：**

- 最多支持 50 个设备或子账号
- 日常个人使用完全足够

**注意：** 在选择计划页面，如果不想添加付款方式，可以直接取消并退出，功能可以正常使用。

![选择免费套餐](/assets/images/post/8beadd8f2958db1fe0ea363e28fe1749.jpg)

## 二、下载并安装 WARP 客户端

### 2.1 找到下载入口

重新进入 Zero Trust，在左下角找到"集成"选项，进入"服务提供商"页面，找到 WARP 客户端注册入口。

点击"添加设备"，系统就会提示你下载 WARP 客户端。

![下载 WARP 客户端](/assets/images/post/cf7622057d74c8e3c7e1d3f5f84173be.jpg)

### 2.2 选择对应系统

客户端支持 Windows、macOS、Linux、iOS、Android 等主流系统，选择对应系统下载稳定版即可。

**Windows 版本：**
- 安装包大小：约 130MB
- 支持系统：Windows 10、Windows 11、Windows 365

**macOS 版本：**
- App Store 下载

![选择系统版本](/assets/images/post/cd2116984d54f036aab1e565da04971b.jpg)

## 三、配置 Zero Trust 策略

安装完成后，回到浏览器页面继续下一步设置。

### 3.1 设置注册策略

系统会引导你设置注册策略：

- 默认会自动允许当前登录邮箱注册设备
- 也可以自定义策略名称

![配置注册策略](/assets/images/post/a80e6330b069dd9d7e34da3265de8467.jpg)

### 3.2 选择服务模式

**建议选择：流量和 DNS（推荐）**

这种模式更稳定，双通道运行：

- DNS 出问题可以走流量代理
- 流量异常可以通过 DNS 进行切换

![选择服务模式](/assets/images/post/6dc0a4a0daf0cb626d809298e7466746.jpg)

### 3.3 配置默认路由

在默认路由配置阶段，选择"排除模式"：

- 默认将所有流量发送到 Cloudflare
- 仅对例外流量进行排除

**重要提示：** 这一步一定要关闭电脑上其他 VPN 或代理工具，否则后续无法正常连接。

![配置路由模式](/assets/images/post/832b23a2fad66dcdad71c1b27d8848cf.jpg)

## 四、客户端登录和连接

### 4.1 登录 Zero Trust

全部设置完成后，系统会提示你在客户端登录 Zero Trust：

1. 打开右下角任务栏中的 WARP 图标
2. 进入"设置"→"偏好设置"→"账户"
3. 选择"使用 Cloudflare Zero Trust 登录"
4. 输入刚才创建的团队名称
5. 系统会向注册邮箱发送验证码
6. 填写验证码即可完成登录

![客户端登录](/assets/images/post/46b4dfaa6c01f92a79d93452d04d36dc.jpg)

### 4.2 建立连接

登录成功后，点击"连接"。

几秒钟内就会建立隧道，状态显示：**已连接，您的 Internet 连接受保护**。

## 五、性能测试

连接成功后进行测速：

- 下载速度：基本稳定在 550Mbps 左右
- 表现：非常稳定

**注意：** 网速取决于你本身上网的速度，以上数据仅供参考。

### 5.1 国际互联网访问

在访问包括 ChatGPT 和 Claude 在内的 AI 服务时，可以正常使用，没有出现地区限制问题。

### 5.2 连接状态

连接成功后，在客户端状态页面可以看到：

- API 连接：已连接
- DNS：已连接
- WARP：已连接
- 托管中心节点：通常显示为美国旧金山/洛杉矶

![连接状态显示](/assets/images/post/2576c5ef4554f0ad33b9d214661cf24e.jpg)

### 5.3 流量统计

流量统计显示已使用数 GB，这个免费方案本身并没有流量限制，日常使用基本不用担心额度问题。

## 七、出口节点说明

### 7.1 默认出口

默认出口通道是香港 HKG，它的连接逻辑是性能最优的机房，而不是距离最近的。

可以更换国家和地区，比如日本、韩国等。

### 7.2 专用出口 IP（付费）

如果您需要让流量从特定国家/地区的 IP 地址出去（如访问有地域限制的服务），那就要付费购买 Enterprise 计划。

**操作步骤：**

1. 购买位于目标国家/地区的专用出口 IP（Dedicated Egress IP）
2. 进入 Gateway → Egress Policies
3. 创建规则匹配特定用户组或域名
4. 将流量路由到指定的专用出口 IP

## 八、重要说明

### 8.1 WARP 不是传统 VPN

WARP 不会隐藏您的真实地理位置，而是用一个与您实际位置相近的 Cloudflare IP 替代原 IP。

**这是设计特性，不是 Bug。**

### 8.2 无法解锁地域限制内容

无法通过 WARP 解锁地域限制内容，如 Bing Chat、Netflix 等区域限制服务。

WARP 无法像传统 VPN 那样绕过（因为它会保留您的大致地理位置）。
