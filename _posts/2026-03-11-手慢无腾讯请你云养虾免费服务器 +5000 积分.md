## 01｜注册 + 领 5000 积分

浏览器打开活动页面链接，点击「登录」，微信扫码。

登录完成后，直接打开用量管理页面领 5000 积分。

点击「限时领取」。

5000 积分包直接到账，有效期一年。

5000 积分什么概念？

CodeBuddy 58 元/月的个人套餐，每个月 2000 积分。现在直接送 5000，两个半月的量。

爽用。

划重点，积分在 CodeBuddy 和 WorkBuddy 通用。领完直接在手机上用 Claw 指挥 Agent 干活。

老用户也能领，直接去「用量管理」页面领就行。

## 02｜安装 CodeBuddy + 领免费服务器

回到第一步的活动页面，点右上角「下载」。

系统会自动开始下载 CodeBuddy 客户端。老用户忽略这步。

安装完成后打开，微信扫码登录，引导页一路跳过就行。

进入 CodeBuddy 界面后，随便聊一句。

和 WorkBuddy 一样，CodeBuddy 也支持很多模型。

接下来领免费服务器，开启「云养虾」。

回到活动页面，下拉，找到「立即领奖」，点击它。

系统会在 CodeBuddy 里弹出腾讯云登录页面。

微信扫码登录，没有腾讯云账号的会自动注册一个。

点击「立即领取」。

选「轻量应用服务器 2 核 2G 4M」。

镜像选 Ubuntu22.04-Docker26，其他默认。

点「立即购买」。

0 元支付，不用绑卡，一路点就行。

菜单栏或者用下面这个链接打开腾讯云控制台，就能看到你领取的服务器了。

点击它，查看服务器详情。

先免费一个月。

连续活跃 7 天，0 元再续费 2 个月，最长白嫖 3 个月。

云服务器天然隔离，搞坏了重装就行，不影响主力机器。24 小时在线，手机随时发消息指挥干活。

这台免费服务器刚好够用。

## 03｜三分钟装龙虾

服务器到手，接下来安装 OpenClaw。

有动手能力的继续用这个 Ubuntu 系统，自己装。

新手小白直接重装成腾讯云 OpenClaw 专属系统，零命令行搞定。

这次以重装为例。

在控制台找到你的服务器，点右边的「...」，选「重装系统」。

选「使用应用模板」，找到 OpenClaw(Clawdbot)。

设一个登录密码，点确定。

等几分钟。

回到控制台，你的服务器已经变成「龙虾」了！

点「登录」。

免密登录，终端里输这行命令：

```
openclaw onboard
```

进入初始化配置，选模型、选渠道，按提示走就行。

一行代码没写，你的「云龙虾」就装好了。

具体怎么玩，翻看我之前的 OpenClaw 系列文章。

回到腾讯云控制台，点进「管理实例」，进入「应用管理」页面。

这里有可视化配置面板，模型、渠道、Skills 全都能在这里设置。有手就行。

十几种国产模型、Coding Plan 一键配置。飞书、企微、QQ、钉钉四大国内 IM 渠道也可以在面板里一键接入。

最后教你一个更「骚」的操作。

CodeBuddy 能直接连上你这台腾讯云服务器，在对话框里帮你操作。查看运行状态、部署项目、执行命令、管理防火墙，甚至都不用自己 SSH 登录。
