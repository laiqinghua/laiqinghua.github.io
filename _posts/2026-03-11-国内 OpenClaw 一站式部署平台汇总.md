# 国内 OpenClaw 一站式部署平台汇总 🦞

> 最近很多小伙伴问我：有没有国内可以直接部署 OpenClaw 的平台？不用自己配置环境、不用折腾 API Key 的那种。今天我就来给大家汇总一下！

## 为什么需要一站式部署平台？

说实话，自己部署 OpenClaw 确实有点门槛：

- **环境配置复杂**：Node.js、npm、各种依赖库
- **API Key 获取麻烦**：要注册阿里云、实名认证、开通服务
- **网络问题**：GitHub 下载慢、npm 镜像配置
- **运维成本**：服务器维护、日志监控、故障排查

一站式部署平台就是帮你搞定这些麻烦事，**开箱即用**！

## 部署注意事项

### ⚠️ API Key 配置

无论选择哪个平台，都需要配置大模型的 API Key：

1. **阿里百炼**：https://bailian.console.aliyun.com/
2. **百度文心**：https://cloud.baidu.com/product/wenxinworkshop
3. **腾讯混元**：https://cloud.tencent.com/product/hunyuan
4. **华为盘古**：https://www.huaweicloud.com/product/pangu.html

💡 **建议**：优先选择阿里百炼千问，性价比高，效果好。

### ⚠️ 机器人配置

QQ/微信机器人需要额外配置：

- **QQ 机器人**：使用 NTQQ 协议或 go-cqhttp
- **微信机器人**：使用 WeChatFerry 或企业微信 API
- **Telegram**：直接调用 Bot API

⚠️ **警告**：QQ/微信机器人有封号风险，建议用小号！

### ⚠️ 成本估算

以个人使用为例：

## 快速开始指南

以阿里云 FC 为例：

### 步骤 1：注册阿里云

https://www.aliyun.com/

需要实名认证（必须的）。

### 步骤 2：开通服务

1. 开通函数计算 FC
2. 开通百炼大模型服务

### 步骤 3：获取 API Key

百炼控制台 → API-KEY 管理 → 创建新 Key

### 步骤 4：一键部署

函数计算控制台 → 应用中心 → OpenClaw → 一键部署

### 步骤 5：配置环境变量

```
BAILOU_API_KEY=你的 API-KEY
DEFAULT_MODEL=qwen-plus
QQ_BOT_ENABLED=true
```

### 步骤 6：测试

访问部署后的 URL，发送测试消息。

完成！🎉
